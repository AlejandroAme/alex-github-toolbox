## microsoft/markitdown

GitHub: https://github.com/microsoft/markitdown

### Purpose
Python tool for converting files and office documents into Markdown.

### Why I saved it
Useful for preparing documents for AI workflows, RAG pipelines, documentation automation, and technical text extraction.

### Possible use cases
- Convert PDFs into Markdown
- Convert Word, PowerPoint, and Excel files into Markdown
- Prepare documents for LLM processing
- Build automated documentation pipelines
- Extract readable content from technical documents
